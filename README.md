# Portfolio Website using Flask

## Overview

A simple personal portfolio website built using Flask, HTML, and CSS.

## Features

* Personal portfolio homepage
* Contact form
* Flask routing
* HTML templates
* CSS styling

## Technologies Used

* Python
* Flask
* HTML
* CSS

## Run the Project

```bash
pip install flask
python app.py
```

Open:

http://127.0.0.1:5000
